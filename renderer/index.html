✈️ Black Box (activity timeline)

          

            Passively records a local, scrubbable timeline of your activity — commands,
            messages, meetings, notes, tabs. Stored on this machine only, never sent anywhere.
            Open it with F4 or the ✈️ button in the status bar.

Web extensions (unpacked)

          

            Load an unpacked Chromium extension folder (must contain manifest.json)
            into the web tabs. Electron supports only a subset of extension APIs, so some
            extensions may not fully work. Press F12 in a web tab for its DevTools.
          

          

          

            
            
            
          

        


        

          
Automation (CDP remote debugging)

          

            Lets external tools (e.g. Claude Code via Chrome DevTools Protocol) drive this app.
            Local-only (127.0.0.1). Blank = off. Restart required. Anything on this machine
            can attach while enabled.
          

          
          
          

            
            
          

        


        

          
Quick commands

          

            One per line. Run them into an SSH tab from that tab's hover menu.
          

          

            
          

          

            
            
          

        


        

          
SSH auto-hop commands

          
          
          

            These ssh … commands are auto-run after an SSH tab reconnects (one per
            line) when auto-hop is enabled. Edit or delete lines to fix them, then Save.
          

          

          

            
            
          

        


        

          
ssh:// links

          
          
          
          
          
          

            Clicking an ssh://user@host:port link opens a new terminal and connects.
            If a jump host is set, Cockpit connects there first and then runs
            ssh user@host to reach the target. Pick a saved SSH session above to
            use it as the jump host, or type one below. Auth uses the SSH agent (or your
            ~/.ssh keys).
          

        


        

          
Project accounting

          
          

            Assign a project to a tab from its hover cheat-sheet ("Track time as"). With this on,
            activating that tab switches the running timer to its project. Tabs with no assigned
            project leave the current one running.
